プライバシーポリシー

私たちは、お客様のプライバシーを尊重し、個人情報の保護に努めています。このプライバシーポリシーは、Habitaアプリ（以下「本アプリ」）がどのような情報を収集し、どのように使用するかを説明します。

1. 収集する情報

1.1 ユーザーが提供する情報

  • 習慣の名前、アイコン、色、リマインダー時間
  • 習慣の完了状況と履歴
  • メモやコメント

1.2 自動的に収集される情報

本アプリは、使用状況の分析や機能改善のための情報を自動的に収集することはありません。

2. 情報の使用目的

収集した情報は以下の目的でのみ使用されます：

  • 習慣の追跡と管理機能の提供
  • リマインダー通知の送信
  • 統計情報の表示
  • iCloud同期によるデバイス間のデータ共有

3. データの保存と同期

3.1 ローカルストレージ

すべての習慣データは、お客様のデバイスにローカルに保存されます。

3.2 iCloud同期

iCloudを有効にしている場合、データは暗号化されてAppleのiCloudサーバーに保存され、お客様の他のデバイスと同期されます。iCloudの利用には、Appleのプライバシーポリシーが適用されます。

4. 第三者との情報共有

本アプリは、お客様の個人情報を第三者と共有、販売、または貸与することはありません。すべてのデータはお客様のデバイスとiCloudアカウント内に保持されます。

5. データの削除

お客様は、いつでも以下の方法でデータを削除できます：

  • 個別の習慣を削除する
  • アプリをアンインストールする（ローカルデータが削除されます）
  • iCloudから本アプリのデータを削除する（設定 > Apple ID > iCloud > ストレージを管理）

6. セキュリティ

お客様のデータは以下の方法で保護されています：

  • デバイス上のデータは、iOSのセキュリティ機能により保護されます
  • iCloud同期時は、Appleの暗号化技術により保護されます

7. 子供のプライバシー

本アプリは、13歳未満の子供から故意に個人情報を収集することはありません。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、アプリ内でお知らせします。
